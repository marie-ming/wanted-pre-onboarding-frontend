# 원티드 프리온보딩 프론트엔드 - 사전 과제

원티드 프리온보딩 프론트엔드 과정 사전 과제 제출을 위한 레파지토리입니다.

## 프로젝트 정보
- 프로젝트명 : To Do
- 팀원 : 마리/정민지
- 배포 링크 : [https://phenomenal-daifuku-64a2cf.netlify.app](https://phenomenal-daifuku-64a2cf.netlify.app)

## 프로젝트 내용

- `/signin` 로그인
- `/signup` 회원가입
- `/todo` 투두리스트
  - 투두 등록, 조회, 수정, 삭제

## 프로젝트 실행

1. `npm install`
2. `npm start`
